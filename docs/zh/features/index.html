功能说明

“音乐游戏培训” app是一款练习音乐游戏的应用。
你可以通过播放视频查看音符，也可以直接触摸视频模拟演奏。

添加播放视频

有四种方法可以添加视频在应用中播放。

Camera Roll

“Camera Roll” 是您在设备上录制视频时保存的部分。你可以从 “照片” 应用中查看，例如。
下面是如何添加它。

  1. 打开 “视频列表” 画面
  2. 点击屏幕右下方的 “添加视频” 按钮
  3. 从附加建议列表中选择 “Camera Roll”
  4. 选择你要添加的视频
  5. 输入您的注册信息并点击 “加法” 按钮

如果您在这一步骤后返回 “视频列表 “屏幕，您将看到您添加的视频列表。

iCloud Drive

iCloud Drive是由 “Apple” 提供的在线存储服务。你可以在 “文件” 应用中查看。
下面是如何添加它。

  1. 打开 “视频列表” 画面
  2. 点击屏幕右下方的 “添加视频” 按钮
  3. 从附加建议列表中选择 “iCloud Drive”
  4. 选择你要添加的视频
  5. 输入您的注册信息并点击 “加法” 按钮

如果您在这一步骤后返回 “视频列表 “屏幕，您将看到您添加的视频列表。

从个人电脑传输

这是一种将视频从PC上的音乐应用（原iTunes）传输到设备的应用存储的方法。
“使用 iTunes 在电脑与 iOS 或 iPadOS 设备之间共享文件” 页面会告诉你如何将其转移到App中，请参考此页面。
下面是如何添加它。

  1. 打开 “视频列表” 画面
  2. 点击屏幕右下方的 “添加视频” 按钮
  3. 从附加建议列表中选择 “应用内”
  4. 选择你要添加的视频
  5. 输入您的注册信息并点击 “加法” 按钮

如果您在这一步骤后返回 “视频列表 “屏幕，您将看到您添加的视频列表。

播放YouTube视频

YouTube是谷歌的视频分享服务。存在别人上传的音乐游戏视频。
下面是如何添加它。

  1. 打开 “视频列表” 画面
  2. 点击屏幕右下方的 “添加视频” 按钮
  3. 从附加建议列表中选择 “YouTube”
  4. 从YouTube网站上打开你要添加的视频页面
  5. 点击视频页面右下角的 “添加视频” 按钮
  6. 输入您的注册信息并点击 “加法” 按钮

如果您在这一步骤后返回 “视频列表 “屏幕，您将看到您添加的视频列表。

因为它不下载YouTube视频，所以每次在这个应用上播放视频都会产生流量。

伪游戏玩法

当您在播放视频时，点击视频不会暂停。
所以，你可以玩一个伪音乐游戏。

还有很多其他功能可以帮助你练习。

  • 调整视频播放速度
    您可以在0.25-2.00倍速度范围内调整播放速度
  • 调整视频播放的开始/结束位置
    你可以设置游戏范围的开始和结束，以专注于游戏中最难的部分
  • 放大/缩小并调整视频的位置
    可以进行缩放和定位，使视频适合设备的分辨率
  • 反转视频画面
    视频画面可以从左到右翻转
  • 支持视频屏幕的纵向和横向方向
    您可以在纵向或横向模式下固定您的设备方向
  • 在播放過程中，可倒退/快進幾秒鐘的快捷鍵
    不需要暫停和操縱時間

收藏播放位置

有一个功能可以为视频播放的开始/结束位置做书签。
你可以为每个数字设置一个开始/结束的位置，这样你就可以快速改变你要练习的部分的播放位置。

支持各种游戏应用中的触摸线

每个设备型号的音乐游戏应用往往有不同的触线。触摸线在设备上的位置可能与您正在寻找的视频不同。
为了解决这个问题，这款应用支持引导显示，帮助你调整到设备上的决策线。
